# ギフテッドスペース LP サイト

市川市で活動する、生きづらさを抱える若者と家族のためのサードプレイス「ギフテッドスペース」の公式ランディングページです。

## 🌟 概要

> **「土曜日、家族以外のだれかと話せる場所。」**

ギフテッドスペースは、生きづらさを抱える若者とそのご家族が、自分らしくいられる居場所を提供するNPO法人（設立準備中）です。

## 🛠️ 技術スタック

- **HTML5** - セマンティックな構造
- **CSS3** - Tailwind CSS (CDN) + カスタムスタイル
- **JavaScript** - Vanilla JS
- **アイコン** - Lucide Icons
- **フォント** - Noto Sans JP (Google Fonts)

## 🚀 ローカルでの確認方法

1. リポジトリをクローン
2. `index.html` をブラウザで開く
