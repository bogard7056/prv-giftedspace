# ギフテッドスペース ストーリーマップ & ナラティブデザイン

## 概要

このドキュメントは、ギフテッドスペースのランディングページにおける物語構造、感情設計、日本語コピーの全体設計書である。ターゲットは（1）生きづらさを抱える若者本人、（2）その家族の二軸。ページを通じて「認識 → 共感 → 希望 → 行動」の感情アークを辿らせる。

---

## 1. ヒーローナラティブ — 最初の3秒で心を掴む

### 設計思想

ヒーローは「説明」ではなく「承認」から始める。訪問者の多くは、検索を重ね、いくつものサイトを見て疲弊している。だからこそ、最初の一文で「ここは自分のための場所だ」と感じてもらう必要がある。売り込みではなく、静かな肯定。

### ヘッドライン候補

**案A（現行ベース・推奨）**
```
土曜日、
家族以外のだれかと
話せる場所。
```
- 評価: 具体的（土曜日）、感覚的（話せる）、余白がある。現行のまま維持を推奨。この一行で「何をする場所か」が伝わりつつ、押しつけがましさがない。

**案B（問いかけ型）**
```
ひとりの時間が、
長すぎると感じたら。
```
- サブヘッドライン: 「毎週土曜日、市川に静かで温かな居場所があります。」
- 評価: 当事者の内面に直接語りかける。「長すぎる」という表現が、引きこもりや孤立の時間感覚に寄り添う。

**案C（安心感先行型）**
```
うまく話せなくても、
ここにいていい。
```
- サブヘッドライン: 「生きづらさを抱える若者と家族の、土曜日の居場所。」
- 評価: 来ること自体のハードルを下げる。「話せなくてもいい」という許可を最初に出す。

**案D（家族向け寄り）**
```
「あの子の居場所が
見つからない」と
思ったことは
ありませんか。
```
- サブヘッドライン: 「ギフテッドスペースは、ご本人とご家族が安心できるサードプレイスです。」
- 評価: 親の検索動機に直接応答する。ただし当事者本人にはやや距離がある。

### サブヘッドライン候補

```
【汎用・推奨】
どんな人にも、可能性がある。
生きづらさを抱える若者と家族が、
自分らしくいられる、静かで温かな居場所。

【当事者寄り】
PCを触ってみるもよし、ただ座っているだけでもよし。
あなたのペースで過ごせる、土曜日の3時間。

【家族寄り】
お子さまのこと、おひとりで抱えていませんか。
まずはご家族だけのご相談からでも大丈夫です。
```

### トラストインジケーター（ヒーロー直下）

現行の3点を維持しつつ、文言を微調整:
```
専門機関連携 ── 当事者が運営 ── 家族もサポート
```
「専門家連携」→「専門機関連携」で具体性を上げる。「当事者目線」→「当事者が運営」で差別化を明確にする。

## 2. ストーリーアーク構造 — ページ全体の感情設計

### 感情曲線

```
感情の高さ
  ↑
  │          ③ 転機          ⑤ 未来
  │         ╱  ╲           ╱
  │        ╱    ╲    ④   ╱
  │       ╱      ╲  活動 ╱
  │      ╱        ╲  ╱╲╱
  │  ② 共感        ╲╱
  │   ╱
  │  ╱
  │ ╱
  │╱ ① 承認
  └─────────────────────────────→ スクロール
```

### 第1幕: 承認（Hero）— 「あなたを見ている」

**目的**: 訪問者が「ここは自分に関係がある」と感じる

**感情**: 安堵、被受容感

**設計**:
- ヘッドラインは声のトーンで。宣言ではなく、そっと差し出す。
- ビジュアルは明るすぎない。柔らかい光、少人数、自然な空気感。
- CTAは2つ：「活動に参加してみる」（行動準備ができている人向け）と「相談・お問い合わせ」（まだ迷っている人向け）。

---

### 第2幕: 共感（Problem / Empathy）— 「わかっている人がいる」

**目的**: 訪問者の痛みを代弁し、「ここの人たちは本当に理解している」と思わせる

**感情**: 共感、驚き（「自分の気持ちを言い当てられた」）

**セクション見出し候補**:
```
【推奨】
「一人で抱え込んでいませんか？」

【代替案】
「外に出るのが怖い日も、あっていい。」

【家族向け強化版】
「一人で悩んでいるのは、あなただけではありません。」
```

**本文コピー（推奨版）**:
```
外に出るのが怖い。将来が見えない。
誰かに話したいけれど、きっかけがわからない。

——あるいは、家族として、
「この子の将来はどうなるのだろう」と
眠れない夜を過ごしていませんか。

その気持ちを、私たちはそのまま受け止めます。
なぜなら、私たち自身が同じ道を歩いてきたからです。
```

**設立者体験カード（リライト）**:
```
高校卒業後、20代前半を誰とも交流なく過ごしました。
外に出ることも、誰かに相談することもできなかった。
居場所を探すことすら、当時の自分には困難でした。

だからこそ、今あなたが感じている
「不安」や「どこにも行けない」という気持ちに、
心から寄り添いたいと思っています。
```

**設計ポイント**:
- 当事者と家族の両方に語りかける。現行は当事者寄りなので、家族への一文を追加。
- 「私たち自身が同じ道を歩いてきた」で、このあとの設立者ストーリーへの伏線を張る。

---

### 第3幕-前半: 活動紹介（Programs）— 「具体的に何ができるのか」

**目的**: 抽象的な「居場所」を具体的な体験として見せる

**感情**: 好奇心、安心（「これなら自分にもできるかも」）

**セクション見出し候補**:
```
【推奨】
できることは、ここにある。

【現行】
ギフテッドスペースの活動

【代替】
土曜日の3時間で、できること。
```

**サブコピー候補**:
```
【推奨】
難しいスキルは必要ありません。
あなたの「やってみたい」が、ここでは一番大切です。

【現行ベース】
最新の技術は、あなたの「表現」を助ける優しい道具です。
```

**各アクティビティのリライト**:

**AIバイブコーディング**:
```
見出し: AIバイブコーディング
本文: プログラミングの知識は一切不要。
「こういうものがあったらいいな」という気持ちを
AIに伝えるだけで、実際に動くツールやアートが生まれます。
あなたの感性が、そのまま作品になる体験です。
```

**ホームページ作成**:
```
見出し: ホームページ作成
本文: 自分だけのページを、インターネット上に持つ。
好きなことを発信したり、ポートフォリオを作ったり。
デジタルの世界に「自分の居場所」を築く楽しさを、
一緒に体験しましょう。
```

**当事者同士の談話**:
```
見出し: 当事者同士の談話
本文: PCに向かうだけが活動ではありません。
お茶を飲みながら、ゆったりと話す時間も。
同じような経験を持つ仲間との、
無理のない交流がここにはあります。
```

**ブリッジカード（大型カード）見出しリライト**:
```
【推奨】
「挨拶の練習から始めた人が、
ここにはいます。」

【現行】
「挨拶の練習から、ゆっくりで大丈夫。」
```

「ここにはいます」と変更することで、設立者自身の経験と重ね、実在の物語として伝える。

---

### 第3幕-後半: 設立者ストーリー（Our Story）— 感情のクライマックス

**目的**: ページ全体の感情的頂点。「この人が作った場所なら信じられる」

**感情**: 感動、信頼、希望

**セクション見出し候補**:
```
【推奨】
私自身の変化は、
ある一つの場所から
始まりました。

【代替案A】
「あの日、はじめて
誰かと話せた場所。」

【代替案B】
一人の経験が、
この場所になりました。
```

**本文コピー（リライト推奨版）**:

```
25歳の時、ふなばし地域若者サポートステーションを訪れたことが、
私の人生の転機でした。

高校を卒業してから、ずっと一人でした。
就労経験もなく、社会との接点を持てないまま、
20代前半の数年間を過ごしました。

サポートステーションで出会った仲間たちも、
同じように就労経験に乏しい人たちでした。
挨拶の練習から始まった日々。
最初は「おはようございます」の一言すら緊張した。
でも、その小さな一歩が、すべての始まりでした。

やがて私は、自分の経験を活かして
同じ境遇の人たちの力になりたいと考えるようになりました。

しかし、当事者として体験を語っても、
「うちの子とは状況が違う」という反応が多かった。
そこで気づいたのです。

必要なのは、誰かの体験談ではなく、
「心理的安全性のある居場所」そのものだと。

言葉で励ますのではなく、
安心して過ごせる場所を作ること。
PCやAIでのものづくり、何気ない談話を通じて、
人と人がゆるやかにつながれる場所を。

それが、ギフテッドスペースの原点です。
```

**設計ポイント**:
- 段落を細かく区切り、一段落一感情で進める。
- 時系列に沿って「孤立 → 転機 → 挫折 → 気づき → 決意」の5段構成。
- 現行の文章は圧縮されすぎており、感情の起伏が足りない。展開を丁寧にする。

## 3. 今後の展望セクション — 「共鳴する社会」への道筋

### セクション設計

このセクションは現行サイトに存在しない。設立者ストーリーとCTAの間に新設する。「この団体には未来がある」と感じさせ、信頼性と期待感を同時に醸成する。

### セクション見出し

```
【推奨】
今後の展望 ── 共鳴する社会へ

【サブコピー】
一つの居場所から始まった活動を、
やがて社会全体の「つながり」へ。
```

### ビジョンステートメント

```
私たちが目指すのは、「共鳴する社会」です。

一人の小さな声が、もう一人の心に届く。
その共鳴が、やがて社会全体を変えていく。

ギフテッドスペースは、その最初の一音を
この市川の場所から鳴らし始めます。
```

### ロードマップ（具体的マイルストーン）

```
Phase 1 ── 土台を築く（現在〜2026年度）
  ・NPO法人格の取得
  ・市川市との連携協定の締結
  ・毎週土曜日の定期活動の安定運営
  ・参加者コミュニティの形成

Phase 2 ── 活動を広げる（2027年度〜）
  ・平日プログラムの開設（個別相談・少人数ワークショップ）
  ・オンライン参加の仕組みづくり（外出が難しい方のために）
  ・家族向けプログラムの充実（保護者同士のピアサポート）
  ・活動成果の可視化と発信

Phase 3 ── つながりを紡ぐ（2028年度〜）
  ・近隣市（船橋・松戸・浦安）への活動展開
  ・他の若者支援団体とのネットワーク構築
  ・企業との連携による就労体験プログラム
  ・当事者が支援者になる「ピアサポーター育成」
```

### 展望セクションの日本語コピー

```html
見出し: 今後の展望
サブ見出し: 一つの居場所から、共鳴する社会へ。

導入文:
ギフテッドスペースの活動は、まだ始まったばかりです。
しかし、私たちには明確なビジョンがあります。
一人ひとりの小さな変化が、やがて社会を動かす力になる。
そんな「共鳴する社会」を、この場所から実現していきます。

マイルストーン見出し1: 土台を築く
マイルストーン本文1:
NPO法人格の取得と市川市との連携を通じて、
毎週土曜日の活動を安定的に継続できる基盤を作ります。

マイルストーン見出し2: 活動を広げる
マイルストーン本文2:
オンライン参加や平日プログラムの開設により、
より多くの方が自分に合った形で参加できる環境を整えます。
保護者同士のピアサポートの場も設けていきます。

マイルストーン見出し3: つながりを紡ぐ
マイルストーン本文3:
近隣地域への展開と、支援団体・企業との連携を通じて、
当事者が支援者へと成長できる循環を生み出します。

締めの一文:
どんなに小さな一歩も、誰かの一歩とつながっている。
その連鎖こそが、「共鳴する社会」の始まりです。
```

## 5. ソーシャルプルーフフレームワーク — 実績がまだない段階の信頼構築

### 設計方針

実際の参加者の声がまだ集まっていない段階では、「テスティモニアル」ではなく「信頼の構造」で代替する。嘘のない範囲で、信頼を積み上げる要素を配置する。

### フレームワーク1: パートナーシップの可視化（現行を強化）

```
見出し: 連携・協力
配置: 設立者ストーリーの直後

要素:
  ・市川市（連携予定）—— 自治体のロゴまたは名称
  ・ふなばし地域若者サポートステーション —— リンク付き
  ・NPO法人設立準備中 —— 進捗の透明性

コピー:
「地域の専門機関と連携しながら、
安心できる支援体制を整えています。」
```

### フレームワーク2: 数字で見せる（活動実績が出てきたら更新）

```
見出し: なし（信頼バーとしてさりげなく配置）

初期段階:
  ・活動開始: 2025年〜
  ・開催頻度: 毎週土曜日
  ・参加費: ¥500 / 回
  ・連携機関: 2団体

成長段階（数字が出てきたら更新）:
  ・累計参加者数: ○○名
  ・活動回数: ○○回
  ・リピート率: ○○%
```

### フレームワーク3: 「声」のプレースホルダー（将来の参加者の声用）

テスティモニアルが集まるまでの間、以下の構造を準備しておく。
現時点ではこのセクションは非表示にしておき、実際の声が2〜3件集まった段階で公開する。

```
デザイン構造（非公開テンプレート）:

カードレイアウト:
  ・イニシャルアイコン（A.S. のような匿名表記）
  ・属性タグ（例: 20代・参加者 / 保護者）
  ・本文（2〜3行の短い感想）

```

### フレームワーク5: 心理的安全性の「見える化」

テスティモニアルの代わりに、参加時の約束事を明示することで信頼を構築する。

```
見出し: 私たちの約束

・無理に話す必要はありません
・途中参加・途中退室は自由です
・他の参加者の情報は外に持ち出しません
・「できない」ことを責める人はいません
・あなたのペースが、ここでは正解です
```

このセクションは独立セクションとしても、FAQ内の補足としても配置可能。「約束」という形で提示することで、テスティモニアルなしでも「ここは安全な場所だ」と感じてもらえる。

## 7. 実装上の注意

### トーンガイドライン
- **使う**: 柔らかい語尾（〜ます、〜ません）、問いかけ、余白のある表現
- **避ける**: 命令形、感嘆符の多用、「絶対」「必ず」などの断定語、業界用語（エンパワメント等）
- **特に避ける**: 過度にポジティブな表現。「大丈夫！」「きっとできる！」のような無責任な励ましは、当事者にとって逆効果になりうる

### デュアルターゲットの扱い
- 本文コピーは基本的に「当事者本人」に語りかける
- 家族向けのメッセージは、カードや補足文として明示的に分離する
- 両者が混在すると、どちらにも響かなくなる

### 文字数の目安
- ヒーロー見出し: 20文字以内
- セクション見出し: 25文字以内
- 本文段落: 1段落80文字以内（モバイルでの可読性）
- CTAボタン: 10文字以内
